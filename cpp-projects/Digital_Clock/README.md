# Digital Clock

This is a simple digital clock program written in C++ language. It displays the current time in hours, minutes, and seconds format on the console.

## Features

- Displays the current time in hours, minutes, and seconds.
- Updates the time continuously, showing the real-time clock.
- Simple and easy-to-understand code.

## Prerequisites

To run this program, you need to have the following:

- C++ compiler (such as GCC or Clang)
- C++ development environment
