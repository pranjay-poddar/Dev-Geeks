<h1>ALGORITHM LEARNER</h1>
<h2>DESCRIPTION</h2>
The project helps users understand various DSA algorithms.

Algorithms included-:

Sorting Algorithms-:

i. Bubble Sort

ii. Insertion Sort

iii. Selection Sort

Searching Algorithms-:

i. Linear Search

ii. Binary Search
