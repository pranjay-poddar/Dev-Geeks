# Flutter Quiz App

A simple quiz app built with Flutter.

## Features

-   Multiple choice questions
-   Correct and incorrect answers
-   Score keeping

## Getting Started

To get started, clone this repository and install the dependencies:

```
flutter pub get
```

Then, run the app:

```
flutter run
```
