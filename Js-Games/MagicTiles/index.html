Game Instructions:
Input the number of seconds you want the timer to run for and click on Start Game.
Leave blank for 10 seconds.

Keys '1', '2', '3' and '4' on your keyboard are to be used to tap the tiles.
Note the black tile in the last row (row 4) and press accordingly.
1 for the leftmost, 2 for the second one, 3 for the third one and 4 for the rightmost.

Check your reflexes!
