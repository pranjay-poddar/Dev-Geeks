<h1>Pythagorean Triplet</h1>
<h2>Tech Stack</h2>

HTML

CSS

JavaScript

<h2>Description</h2>
This project creates a simple web page where users can enter a range and click the "Generate Triplets" button to find and display all the Pythagorean triplets within that range.
