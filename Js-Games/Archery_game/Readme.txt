## Archery Game
This is a simple archery game built using HTML, CSS, and JavaScript. The game allows players to aim and shoot arrows at targets, earning points for hitting the bullseye. It provides a fun and interactive experience for users who enjoy archery or simple browser games.

## Features
Target Shooting: Players can aim and shoot arrows at targets displayed on the screen.
Scoring System: The game keeps track of the player's score based on their accuracy.
Timer: Each level has a time limit within which the player needs to score as many points as possible.
Sound Effects: The game features sound effects to enhance the gaming experience.
Responsive Design: The game is designed to be responsive and playable on various devices and screen sizes.
