# Prime number

## Techstack used
C 

## Details
user is expected to input a number(positive integer) and the program determines if it is a prime number or not.  
returns the divisors of the number.

Examples:  

57;  
Divisors of 57: 1 3 19 57  
57 is not prime

17  
Divisors of 17: 1 17  
17 is prime
